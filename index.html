Главная

В большинстве систем Linux для его запуска используется одно и то же сочетание клавиш по умолчанию: Ctrl-Alt-T.

Для того чтобы вставить содержимое в терминал используется одно и то же сочетание клавиш по умолчанию: Ctrl-Shift-V.

Основные команды

Cd 'change directory` - Команда, которая может изменить рабочий каталог. Попробуйте ввести следующее:

                    
                        
┌──(guest@guest)-[~]

                        
└─$ cd /

                        
┌──(guest@guest)-[/]

                        
└─$

Здесь мы поменяли директорию (папку), в которой мы находимся поменялось с user на корневую директорию.

Также диреторию, в которой мы находимся показывается в квадратных скобках.

                    
                        
┌──(guest@guest)-[/]

                        
└─$ cd ./home

                        
┌──(guest@guest)-[/home]

                        
└─$

Мы можем использовать символ . , для показания текущей директории, от которой мы идём.

Чтоб вернутся в домашную директорию нужно использовать символ ~ .

                    
                        
┌──(guest@guest)-[/home]

                        
└─$ cd ~

                        
┌──(guest@guest)-[~]

                        
└─$

Для просмотра содержимого директории мы используем ls 'list' - выводит нам все элементы

                    
                        
┌──(guest@guest)-[~]

                        
└─$ ls

                        
Видео    Изображения    'Рабочий стол'    Документы    Музыка

                        
Шаблоны    Загрузки    Общедоступные

Нам вывело содержимое нашей домашней директории.

Для просмотра содержимого директории с скрытыми файлами и с правами доступа - мы используем ls -la 

                    
                        
┌──(guest@guest)-[~]

                        
└─$ ls -la

                        
итого 268

                        
drwx------ 22 guest guest  4096 дек 13 21:10  .

                        
drwxr-xr-x  3 root   root    4096 июл 15 20:58  ..

                        
...

Мы можем узнавать о дополнительных функциях команд при помощи данной команды + --help

                    
                        
┌──(guest@guest)-[~]

                        
└─$ ls --help

                        
Использование: ls [ПАРАМЕТР]… [ФАЙЛ]…

                        
Выдаёт информацию о ФАЙЛАХ (по умолчанию о текущем каталоге).

                        
Сортирует в алфавитном порядке, если не задан ни --sort, ни один из параметров -cftuvSUX.

                        

                        
Аргументы, обязательные для длинных параметров, обязательны и для коротких.

                        
     -a, --all                  не скрывать файлы начинающиеся с ...
